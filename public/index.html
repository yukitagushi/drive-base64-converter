Gemini File Search を、白と黒の美しいワークスペースで。

Gemini Lounge は、File Search ストアの作成からファイルの取り込み、ログの確認までをシームレスに統合したコンソールです。 ChatGPT のような没入感あるフローを保ちながら、SaaS チームが日常的に利用できる洗練された UI を目指しました。

  • 公式 API に準拠したストア生成・ファイル連携
  • ZIP / PDF / Office / 画像などバイナリを一元管理
  • Supabase 認証と Gemini API の状態を常時監視
